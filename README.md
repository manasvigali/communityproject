# InvestEd (Jekyll)

This is a Jekyll version of the InvestEd site. Files provided here will render the site with a simple Jekyll layout and include the custom CSS/JS under `assets/`.

Quick steps to add & publish:
1. Create a branch, e.g. `jekyll-site`.
2. Add the files and commit.
3. Push the branch and open a PR to merge to `gh-pages` (or `main`) — GitHub Pages will serve the site from the target branch.
4. If using the provided Actions workflow, ensure the repository branch for Pages is `gh-pages`, then the workflow will build and deploy to Pages when you push to `gh-pages`.
